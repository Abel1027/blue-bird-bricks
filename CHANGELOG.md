# Changelog

## 1.0.0

### Added
- Initial release of Blue Bird Bricks collection
- `blue_bird_flutter_project` brick for generating Flutter applications
  - Multi-package workspace structure with Melos
  - Core package for utilities and localization
  - Design System package with Atomic Design components
  - Routing with go_router
  - Dependency injection with get_it and injectable
  - Internationalization support
  - Theme management
  - Pre-configured Melos scripts
- `blue_bird_flutter_package` brick for generating Flutter packages
  - Clean Architecture (Presentation, Domain, Data layers)
  - BLoC pattern with Cubit for state management
  - Repository pattern with local/remote datasources
  - Dependency injection
  - Freezed for immutable entities
  - Comprehensive test setup (unit, widget, golden tests)
  - Mapper pattern for data transformation
- Comprehensive documentation for both bricks
- MIT License
- README with usage instructions
